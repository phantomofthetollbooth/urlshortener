Url Shortener

Example Creation Usage:

https://shrls.herokuapp.com/add/https://www.google.com

https://shrls.herokuapp.com/add/http://foo.com:80

Example Creation Output

{ "original_url":"http://foo.com:80", "short_url":"https://shrls.herokuapp.com/8170" }

Usage

https://shrls.herokuapp.com/1235

Will redirect to:

https://www.wikipedia.org
